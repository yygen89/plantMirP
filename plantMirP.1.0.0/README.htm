About This Program

MicroRNAs (miRNAs) are a predominant type of small non-coding RNAs (sRNAs) approximately 21 nucleotides in length that play an essential role at the post-transcriptional level by either RNA degradation, translational repression or both through an RNA-induced silencing complex. Identification of these molecules can aid the dissecting regulatory functions. The secondary structures of plant pre-miRNAs are much more complex than that of animal pre-miRNAs. In contrast to prediction tools for animal pre-miRNAs, more less efforts have been contributed to plant pre-miRNAs. A set of novel knowledge-based energy features, which has very high discriminatory power, is proposed and incorporated with existing features for speciﬁcally distinguishing the hairpins of real/pseudo plant pre-miRNAs. A promising performance of area under receiver operating characteristic curve (AUC) of 0.94444 indicates that 5 knowledge-based energy features have very high discriminatory power. The 10-fold cross-validation (CV) result demonstrates that plantMirP with full features has a promising sensitivity of 93.55% and a specificity of 71.61%. Furthermore, plantMirP can greatly balance sensitivity and specificity for real/pseudo plant pre-miRNAs. Taken together, we developed a promising SVM-based program, plantMirP, for predicting plant pre-miRNAs by incorporating knowledge-based energy features.

 

Training model

Before prediction pre-miRNAs, a SVMs model should be constructed with the following command:

perl plantMirP_training.pl -P genuine_premirna.fa -N negData.fa

 

Prediction

perl plantMirP.pl [options]

options:

-D The result directory
-T The fasta file containing test sequences
-P The file containing statistical potentials
-U The file containing scaled model for SVMs
-S The file containing SVMs model
-K The fasta file containing positive sequences
-G The fasta file containing negative sequences
-L The absolute path to the Libsvm binary
-J If this option is set to 1, then using only knowledge-based energy features for prediction (default: 0). Note: the selected feature sets must match corresponding models.
-h Print help information

example:

perl plantMirP.pl -T test.fa -S ./model/svm_model.txt -U ./model/scaled_model.txt -P ./model/potential.txt -D result -L /home/yyg/libsvm/

Or

perl plantMirP.pl -T test.fa -S ./model/svm_model.txt -U ./model/scaled_model.txt -K positiveData.fa -G negativeData.fa -D result -L /home/yyg/libsvm/

 

Prerequisites

plantMirP are implemented in Perl v5.20.2. Currently, only local stand-alone packages of LINUX system will be provided.

Please ensure LIBSVM (Version 3.14) are installed before proceeding.

You can download the installation file of LIBSVM software from http://www.csie.ntu.edu.tw/~cjlin/libsvm/.

Then follow the installation prompts of LIBSVM software to finish installation.

 

Installation

Unzip the installation file and add the directory to your path.
